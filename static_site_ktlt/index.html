An Introduction

Why Truth Trees?

Unlike truth tables, truth trees represent only the assignments of True to sentences. Starting from the original statement, we ask at each step, “Under what conditions (or interpretations) would this statement be True?” Then, we write down those conditions in a systematic way. At each step, then, we only indicate when a statement would be ‘True’, NOT ‘False’. When we complete a tree, we “read up” through the “branches” for the truth values that would make the statement true. So a big difference between tables and trees is that, while a table begins with showing all possible combinations of truth values, a tree shows those values that would make the statement true (if it is possible for it to be true).
